# 🍔 Food Ordering App

A modern restaurant browsing and cart management application built with **React**,**Redux**, and **TailwindCSS**. Users can discover restaurants, view detailed menus, and manage their shopping cart through a fast and responsive user interface powered by JSON Server.

---

## 🚀 Features

- 🏪 **Restaurant Listing** – Browse all available restaurants on the home page.
- 🍽️ **Menu Exploration** – View restaurant details and full menu on a dedicated page.
- 🛒 **Cart Management** – Add or remove items and manage your order seamlessly.
- 🔔 **Toast Notifications** – Instant feedback on every cart action via React Toastify.
- ⚡ **Redux State Management** – Centralized and predictable state with Redux Thunk.
- 📱 **Fully Responsive** – Clean layout optimized for all screen sizes.
